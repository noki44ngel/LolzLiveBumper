LOLZ FORUM AUTO-BUMP

Автоматическое поднятие тем на Lolz Forum каждые 12 часов.

ФАЙЛЫ:
  config.py        - настройки (токены, ID тем, интервалы)
  bump_service.py  - основной сервис
  database.py      - SQLite хранилище
  bump_data.db     - база данных (создаётся автоматически)
  bump.log         - логи

ЗАПУСК:
  pip install -r requirements.txt
  python bump_service.py

НАСТРОЙКА (config.py):
  API_TOKEN        - токен Lolz API
  TELEGRAM_BOT_TOKEN - токен бота (опционально)
  TELEGRAM_CHAT_ID - ID чата для уведомлений
  THREAD_IDS       - список ID тем для поднятия
  ADMIN_USER_IDS   - кто может юзать /status

ЛОГИКА РАБОТЫ:
  1. При старте синхронизирует темы из config.py с базой
  2. Каждые 2 минуты проверяет какие темы готовы к бампу
  3. Поднимает готовые темы через API (POST /threads/{id}/bump)
  4. После успеха ждёт 12ч + случайные 30-300 сек (jitter)
  5. При ошибке повторяет через 1м/5м/15м/1ч
  6. После 10 ошибок подряд пропускает тему на 12ч
  7. Отправляет отчёт в Telegram после каждого цикла

TELEGRAM КОМАНДЫ:
  /start  - приветствие
  /status - статус всех тем (время до следующего бампа)

ОБРАБОТКА ОШИБОК:
  401 - невалидный токен, остановка сервиса
  403 - кулдаун, берёт next_available_time из API
  429 - rate limit, ждёт 2 минуты

ОСТАНОВКА:
  Ctrl+C или SIGTERM - корректное завершение
